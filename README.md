# ⚖️ Conversational Legal AI (RAG)
**Personal Skill-Building Project - Cardiff University**

A memory-aware Retrieval-Augmented Generation (RAG) system developed to enhance my skills in NLP and end-to-end AI deployment. This tool is designed to analyze and chat with UK Legal documents (specifically the 2026 AI Impact Assessments).

## 🚀 Key Features
- **Conversational Memory:** Uses query re-phrasing logic to handle follow-up questions (e.g., "What about its conclusion?").
- **Dynamic Data Ingestion:** Upload and process PDFs in real-time through the UI.
- **Local Privacy:** Optimized to run on Apple Silicon (M4 MacBook Air) using Ollama for local LLM inference.
- **Gradio 6 Interface:** A modern, mobile-responsive UI accessible via a public URL.

## 🛠️ Technical Stack
- **Model:** Llama 3 (via Ollama)
- **Embeddings:** Nomic-Embed-Text
- **Database:** ChromaDB
- **Frameworks:** LangChain, Gradio 6
